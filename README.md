# Santhe-Connect

Android App Development using GenAI – Hospitality Project

---

# 📌 Project Overview

Santhe-Connect is a hospitality and local discovery Android application designed to help tourists explore authentic Karnataka culture, local food, and weekly village markets called “Santhe.”

The application connects travelers with:
- Local food points
- Traditional markets
- Authentic breakfast centers
- Small local businesses
- Cultural experiences

The app is developed using:
- Kotlin
- Jetpack Compose
- Android Studio

---

# 🎯 Problem Statement

Tourists visiting heritage towns often want to experience authentic local food and weekly markets. However, many small local eateries and village markets are not available on major travel applications.

Santhe-Connect solves this problem by providing a platform to discover:
- Traditional Karnataka food
- Weekly village markets
- Local handmade products
- Community-driven hospitality experiences
